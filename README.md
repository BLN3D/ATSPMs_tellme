# Traffic Signal Performance Dashboard

A Python-based web app that uses multimodal AI to analyze **Purdue Coordination Diagrams (PCD)** and estimate **Arrivals on Green (AoG)** and **Level of Service (LOS)**.

### Setup
1. `pip install -r requirements.txt`
2. Set your API Key: `export GEMINI_API_KEY='your_key_here'`
3. Run: `python app.py`
